# repo-growth

> Measure how fast your repo is growing using cloc

## Install

```sh
npm i -g repo-growth
```

## Usage

**Important** Make sure you've pulled master and your repo is in a clean state.

```sh
repo-growth
```

#### `--start`, `-s`

**Default: Date of first commit**

```sh
repo-growth --start 2017-01
```

#### `--end`, `-e`

**Default: Today**

```sh
repo-growth --end 2018-01
```

#### `--freq`, `-f`

**Default: 30**

```sh
repo-growth --freq 7
```
